ABOUT THE PROJECT

A decision-support platform for fuel prices, not just a number generator.

FuelSense AI combines machine learning, live market signals, and clear explanations to help people understand why fuel prices move — not only what they'll be next.

Mission

Make fuel market intelligence transparent and explainable, so predictions are something people can reason about, not just trust blindly.

Vision

To grow from a two-feature regression model into a full market intelligence suite: live APIs, scenario simulation, and historical analytics in one product.

HOW IT'S BUILT

Architecture overview

Frontend

Django Templates, HTML5, CSS3, and vanilla JavaScript — no framework overhead.

Backend

Python and Django handle routing, views, and orchestration between the UI and the model.

Machine Learning

A Random Forest Regressor trained with Scikit-Learn on crude oil and USD-INR data.

Data Processing

Pandas and NumPy clean and shape historical price data before it reaches the model.

Prediction Engine

Joblib-serialized model loaded once and reused across every prediction request.

API Layer

Django views expose prediction and market data as JSON for the dashboard and predict pages.
